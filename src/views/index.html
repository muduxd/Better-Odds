How to Use the Arbitrage Calculator

What is Arbitrage Betting?

Arbitrage betting (also known as arbing) is a betting strategy that takes advantage of different odds offered by different bookmakers. When the combined probability of all outcomes is less than 100%, there's an opportunity for guaranteed profit.

Using the Calculator

1. Enter your total stake amount (the total money you want to invest)
2. Input the decimal odds for each bet (minimum 2 bets required)
3. Add more bets using the "Add" button if needed
4. The calculator will automatically show if there's a profitable arbitrage opportunity

Example

Let's say you find these odds for a tennis match:
•Player A: 2.10 (Bookmaker 1)
•Player B: 2.05 (Bookmaker 2)
With a $100 total stake, this calculator will show you it's profitable because:
1/2.10 + 1/2.05 = 0.476 + 0.488 = 0.964 (96.4% total probability)

Understanding Results

•Stake: The amount to bet on each outcome
•Payout: Your return if that outcome wins
•Green message: Profitable opportunity (shows expected profit)
•Red message: Not profitable (total probability exceeds 100%)
